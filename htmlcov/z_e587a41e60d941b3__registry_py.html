1"""Static lookup tables for Septentrio Binary Format (SBF) decoding. 

2 

3All tables are derived from official Septentrio documentation. 

4Each entry carries an explicit source citation. Do NOT modify values 

5without updating the corresponding source reference. 

6 

7Primary source 

8-------------- 

9AsteRx SB3 ProBase Firmware v4.14.0 Reference Guide (Septentrio). 

10Matches the firmware series of the production test data (v4.14.4). 

11Abbreviated below as **RefGuide-4.14.0**. 

12 

13Supplementary source 

14-------------------- 

15AsteRx SB3 ProBase Firmware v4.15.1 Reference Guide (Septentrio). 

16Field-level definitions are identical to v4.14.0 for all blocks used here. 

17 

18Frequencies 

19----------- 

20Carrier frequencies are fetched from the ``BAND_PROPERTIES`` class 

21attributes in :mod:`canvod.readers.gnss_specs.constellations` (single 

22source of truth). Accessing them as class variables requires no 

23instantiation and triggers no network or file I/O. 

24 

25BeiDou B2I (signal 29) shares the same carrier frequency as B2b 

26(1207.14 MHz); it is mapped to ``BEIDOU.BAND_PROPERTIES["B2b"]``. 

27GLONASS FDMA signals (8-11) have ``freq=None``; use 

28:func:`~canvod.readers.sbf._scaling.glonass_freq_hz` instead. 

29"""

85def decode_svid(svid: int) -> tuple[str, int]: 

86 """Map a Septentrio internal SVID to ``(system, prn)``. 

87 

88 Parameters 

89 ---------- 

90 svid : int 

91 Septentrio internal satellite identifier (field type ``u1``). 

92 

93 Returns 

94 ------- 

95 system : str 

96 RINEX single-letter system code 

97 (``"G"``, ``"R"``, ``"E"``, ``"C"``, ``"J"``, ``"I"``, ``"S"``, ``"L"``). 

98 prn : int 

99 Satellite PRN number (or GLONASS slot number). 

100 ``0`` for GLONASS with unknown slot (SVID 62). 

101 

102 Notes 

103 ----- 

104 Source: RefGuide-4.14.0, Table 4.1.9, p. 255. 

105 Unknown SVIDs return ``("?", svid)``. 

106 """ 

107 for r in _SVID_RANGES: 

108 if r.lo <= svid <= r.hi: 

109 return r.system, svid - r.prn_offset 

110 return "?", svid

113# --------------------------------------------------------------------------- 

114# Frequency lookup: (system, sbf_band) → pint.Quantity 

115# 

116# Built from gnss_specs constellation BAND_PROPERTIES ClassVars — no 

117# instantiation, no network, no file I/O. 

118# 

119# BeiDou B2I maps to "B2b" in gnss_specs (same carrier: 1207.14 MHz). 

120# GLONASS G3 is CDMA (fixed freq); FDMA G1/G2 are intentionally absent 

121# (their freq is slot-dependent and looked up via glonass_freq_hz()). 

122# L-Band MSS ("LBd") has no fixed carrier and is intentionally absent. 

123# 

124# _X_BP aliases are typed as Any to work around pint's incomplete type stubs 

125# (BAND_PROPERTIES ClassVars resolve to Quantity[Unknown] in Pylance). 

126# Runtime values are pint.Quantity[float] in MHz. 

127# ---------------------------------------------------------------------------

177class SignalDef: # pylint: disable=too-many-instance-attributes 

178 """Definition of a tracked GNSS signal modulation. 

179 

180 Attributes 

181 ---------- 

182 number : int 

183 Signal type index (bits 0-4 of the SBF ``Type`` byte). 

184 signal_type : str 

185 Septentrio signal label, e.g. ``"L1CA"``, ``"E5a"``, ``"B1I"``. 

186 system : str 

187 RINEX system letter of the native constellation for this signal number. 

188 freq : pint.Quantity or None 

189 Carrier frequency fetched from 

190 :data:`~canvod.readers.gnss_specs.constellations` BAND_PROPERTIES. 

191 ``None`` for GLONASS FDMA signals (8-11) — use 

192 :func:`~canvod.readers.sbf._scaling.glonass_freq_hz` — and for 

193 L-Band MSS (signal 23). 

194 rinex_obs : str 

195 RINEX 3 observation code, e.g. ``"1C"``, ``"7Q"``, ``"2I"``. 

196 band : str 

197 Human-readable band label, e.g. ``"L1"``, ``"E5b"``, ``"B1I"``. 

198 code : str 

199 Tracking code letter used in the RINEX obs code, e.g. ``"C"``, ``"I"``. 

200 cn0_no_offset : bool 

201 ``True`` only for signal numbers 1 (GPS L1P) and 2 (GPS L2P). 

202 These use ``C/N0 = raw * 0.25`` (no +10 dB offset). 

203 All other signals use ``C/N0 = raw * 0.25 + 10``. 

204 

205 Notes 

206 ----- 

207 Source: RefGuide-4.14.0, Table 4.1.10, p. 256. 

208 CN0 formula: RefGuide-4.14.0, MeasEpochChannelType1.CN0, p. 261. 

209 Frequencies sourced from :mod:`canvod.readers.gnss_specs.constellations`. 

210 """
